## **Integração WSL com Windows**

Abre a pasta atual no VS Code:
```bash
code .
```

Abre a pasta atual no Windows Explorer:
```bash
explorer.exe .
```

Reinicia o WSL — executado no PowerShell do Windows:
```bash
wsl --shutdown
```
