## Por que Python?

Python é uma linguagem de programação criada para ser **simples de ler, fácil de escrever e poderosa na prática**[^2]. Ela é amplamente utilizada para:

* automatizar tarefas repetitivas do dia a dia.
* analisar e tratar dados.
* integrar sistemas e serviços.
* criar pequenas ferramentas que economizam tempo e reduzem erros.
* dar mais autonomia a equipes administrativas e de gestão.

[^2]: Esta referência do que o Python foi retirada de [sua própria documentação](https://docs.python.org/pt-br/3.14/tutorial/index.html).

No contexto do setor público — e especialmente na SPLOR — Python pode ser um grande aliado para **ganhar eficiência, clareza e confiabilidade nos processos**.

## Por que aprender Python agora?

Se você já passou pela primeira Trilha Dev, provavelmente já percebeu algo importante: tecnologia não é só para quem quer “virar programador”. Ela é uma **ferramenta de trabalho**.

Aprender Python neste momento da jornada significa:

* transformar conhecimento conceitual em soluções práticas.
* deixar de depender exclusivamente de terceiros para pequenas automações.
* entender melhor o que é possível (e o que não é) quando conversamos com equipes técnicas[^3].
* ampliar seu repertório profissional sem abandonar sua área de atuação.

Este curso **não é só sobre formar desenvolvedores de software**, mas sobre capacitar pessoas para resolver problemas reais com mais autonomia.

## Para quem é este curso?

Este curso é para você que:

* trabalha na SPLOR ou em áreas administrativas;
* não é programadora ou programador (e tudo bem!);
* quer entender melhor como a tecnologia pode ajudar no seu dia a dia;
* já participou do **Trilha Dev: Início da Jornada** ou tem curiosidade equivalente;
* acredita que aprender algo novo pode tornar o trabalho mais leve e eficiente.

## O que você vai encontrar nesta trilha

Ao longo da **Trilha Dev: Python**, você vai aprender a:

* entender a lógica básica de um programa;
* ler e escrever código Python de forma gradual e guiada;
* usar Python para resolver problemas simples do cotidiano;
* criar pequenos scripts úteis para o trabalho;
* conectar Python com arquivos, dados e rotinas administrativas.

Tudo isso com exemplos próximos da nossa realidade, linguagem acessível e foco no uso prático.
